1. Установи приложение

iPhone/iPad: App Store → Hiddify (или Streisand / V2Box)
Android: Google Play → Hiddify (или v2rayNG)
Windows: github.com/hiddify/hiddify-app/releases → .exe
Mac: github.com/hiddify/hiddify-app/releases → .dmg

2. Добавь профиль

На телефоне: Открой приложение → + → Scan QR Code → наведи камеру на QR выше
На компьютере: Удали старый профиль если есть → Скопируй ссылку выше → + → Add From Clipboard

3. Настрой (только компьютер)

ОБЯЗАТЕЛЬНО: Settings → Routing → Region → Others
ОБЯЗАТЕЛЬНО: Settings → Inbound → Service Mode → Set System Proxy

4. Подключись

Нажми кнопку подключения → Разреши VPN если спросит → Готово!
Проверь: 2ip.ru (обнови: Ctrl+Shift+R / Cmd+Shift+R)
