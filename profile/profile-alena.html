1. Установи приложение

iPhone/iPad: App Store → AmneziaVPN
Android: Google Play → AmneziaVPN
Windows / Mac: github.com/amnezia-vpn/amnezia-client/releases

2. Добавь ключ

На телефоне: Скопируй ключ выше → Открой AmneziaVPN → вставь в поле «Insert key»
На компьютере: Скопируй ключ выше → Открой AmneziaVPN → вставь в поле «Insert key» → нажми Insert

3. Подключись

Нажми кнопку подключения → Разреши VPN если спросит → Готово!
Проверь: 2ip.ru (обнови: Ctrl+Shift+R / Cmd+Shift+R)
